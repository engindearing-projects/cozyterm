Everything you need to ship

📋

Project tracking

Connects to Jira boards and GitHub repos. Get daily briefs via Telegram, track blockers, and never lose context between sessions.

💡

Coaching mode

Use /coach or /explain for warm, patient explanations of commands, errors, and concepts. Learn by doing, not by reading docs.

⚡

Smart router

Heavy tasks go to Claude. Quick lookups stay local on Ollama for free. The router picks the right model automatically.

🐘

Memory system

SQLite + FTS5 full-text search. Decisions, blockers, and context are captured automatically and recalled when you need them.

🛡

Safety rails

Destructive commands trigger a warning before anything happens. Force pushes, dropped tables, and rm -rf all get flagged first.

🌐

Multi-platform

CLI TUI today. Web dashboard, mobile app, and Telegram bot all connect to the same brain. Your context follows you everywhere.

Get started in 60 seconds

1

Clone & install

git clone the repo, then bun install && bun link from the cli/ directory.

2

Setup

Run cozy init — the setup wizard walks you through configuration.

3

Launch

Type cozy — the TUI opens and you can start chatting with Engie.
