What you get

Everything runs locally on your Mac. No hosted services, no data leaving your machine.

memory

Remembers everything

SQLite database with full-text search. Decisions, blockers, and ticket references get captured automatically from conversations. Search with /memory or save notes with /observe.

routing

Picks the right model

Complex tasks (refactoring, multi-file edits) go to Claude through your existing subscription. Quick stuff (status checks, summaries) runs on Ollama locally for free. The router decides automatically.

tasks

Task panel and todos

Press Shift+Tab to open the task panel. See active tool calls, your todo list, and recent observations at a glance. Manage todos with /todo add and /todo done.

coaching

Explains things clearly

Toggle /coach for warmer explanations. /explain breaks down any concept in plain language first, then shows the technical details.

briefs

Morning updates

Cron jobs check your Jira boards and GitHub activity every morning. You get a formatted brief on Telegram before your first coffee. Afternoon follow-ups catch anything that changed.

diagnostics

Fixes itself

engie doctor checks every service, config file, and directory. Add --fix to auto-restart unhealthy services, create missing paths, and rotate logs.

Get running

1

Clone and install

git clone the repo, then bun install from the cli/ directory.

2

Run the wizard

bun run bin/cozy.mjs init sets up the global engie command, installs Ollama, configures API keys, sets up the MCP bridge, and starts services.

3

Start talking

Type engie to open the TUI, or engie "your question" for a quick answer from the command line.
